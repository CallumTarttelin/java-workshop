Python
  • Interpreted Language

The java main

In java every piece of code you write must be saved in a file with the .java extension, this can be as simple as "app.java". Inside this must be a corresponding class of the same name.

The entry point of all java based applications is known as the "main". 

                    
                    public class App {
                        public static void main(String[] args) {
                            System.out.println("Hello World");
                        }
                    }

This is a small hello world example in java

  • Public - Access modifier, these will be covered later on in OOP

  • Class - Covered in OOP section

  • Void - Specifies that a function does not return a value.

  • Static - Specifies that a function is available at class level rather than a class instance level, will be covered more in OOP.

Types - Non primitives (reference variables)

The following do not directly store values, instead containing references to memory addresses 

  • 
                            // string uses titlecase over camelcase due to being an object
                            // assign through string literal
                            String aString = "This is a test string";
                            // assign using the "new" keyword
                            String anotherString = new String("Another string");
                        
  • 
                            // instantiate an array using fixed length notation
                            int[] intArray = new int[10];
                            intArray[2] = 5;
                            intArray[7] = 25;
                            intArray[9] = 50;
                            // output [0, 0, 5, 0, 0, 0, 0, 25, 0, 50]

                            // instantiate an array using array literal
                            String[] stringArray = {"first", "second", "third", "etc"};
                            // output [first, second, third, etc]
                            // this would raise "array index out of bounds" error, you cannot assign to an index that doesn't exist
                            stringArray[7] = "temp";

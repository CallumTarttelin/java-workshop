Exercise - Fibonacci

Fibonacci - Question

A sequence of numbers where the current number (n) in addition to the previous number (n - 1) creates the next number (n + 1).

The first 6 numbers in this sequence are as follows: 0, 1, 1, 2, 3, 5 

                     
                        package basics;

                        public class FibonacciSample {

                            public static void main(String[] args) {
                                FibonacciSample fibonacciSample = new FibonacciSample();
                                long number = 10;
                                System.out.println(fibonacciSample.fib(number));
                            }

                            // returns the number in the Fibonacci sequence
                            public long fib(long n) {
                                return n;
                            }
                        }

A fibonacci solution 

                    
                        package basics;

                        public class FibonacciSample {

                            public static void main(String[] args) {
                                FibonacciSample fibonacciSample = new FibonacciSample();
                                long number = 10;
                                System.out.println(fibonacciSample.fib(number));
                            }

                            // returns the number in the Fibonacci sequence
                            public long fib(long n) {
                                if (n < 2) {
                                    return n;
                                } else {
                                    return fib(n - 1) + fib(n - 2);
                                }
                            }
                        }

OOP

Object-oriented programming is a programming paradigm based around objects. The objects represent things with fields and methods.

  • Object-oriented programming is a paradigm
  • Java is a popular example of a general purpose object oriented language
  • Java, unlike python, is not a pure OO language because not everything is an object

OOP

Object-oriented programming is a programming paradigm based around objects. The objects represent things with fields and methods.

  • Object-oriented programming is a paradigm
  • Java is a popular example of a general purpose object oriented language
  • Java, unlike python, is not a pure OO language because not everything is an object

Classes, Fields, Methods, and Objects

Classes + Fields + Methods

Cat.java


                            public class Cat {
                            }

Cat.java


                            public class Cat {
                                public String name;
                                public boolean inside;
                            }

App.java


                            public class Cat {
                                public String name;
                                public int age;

                                public boolean isAdult() {
                                    return this.age > 1;
                                }

                                public void rename(String name) {
                                    this.name = name;
                                }
                            }
    Classes usually are named in PascalCase
    Classes by themselves aren't that useful
    Classes can hold fields to show properties of a class
    Classes can also have methods to show actions of a class
    These fields and methods are usually in camelCase
    These work as variables and functions associated with the class

Objects

App.java


                        Cat cat = new Cat();
                        Cat otherCat = new Cat();

                        cat.rename("Cat.");

                        System.out.println(cat.name);
                        // Cat.
                        System.out.println(otherCat.name);
                        // null
    Objects are instances of classes
    Created with the new keyword
    You can then access fields and methods for this instance of the class

Creating objects

Cat.java


                        public Cat(String name, int age) {
                            this.name = name;
                            this.age = age;
                        }
    Objects are instances of classes
    Created with the new keyword

June 2020 Java Workshop

Set Up

Accessing the slides

https://callumtarttelin.github.io/java-workshop

Required Software

  • JDK > 11, demos tested in 11 and 14
  • git
  • IDE, examples will be in vscode, so will sys dev 2

Getting exercises in vscode


							git clone https://github.com/CallumTarttelin/java-workshop.git

							code --install-extension vscjava.vscode-java-pack

							code --install-extension richardwillis.vscode-gradle

							code java-workshop

Running examples from terminal

ctrl+shift+i from inside vscode or in terminal 


							./gradlew project:task

							./gradlew helloworld:run

Getting help with exercises (Optional)

code --install-extension ms-vsliveshare.vsliveshare

Then click the icon on the left and click share now

Send one of us the invite link and we'll try to help you

Other IDEs

If you are using intellij then create a new project from version control with https://github.com/CallumTarttelin/java-workshop.git

The gradle menu should be in the right side of the IDE
